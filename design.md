# Diseño Web App Ferretería

## Identidad Visual

### Colores
- **Primario**: Naranja industrial (#FF6B35) - energía y confianza
- **Secundario**: Gris carbón (#2C3E50) - profesionalismo y solidez
- **Acento**: Verde hierro (#27AE60) - disponibilidad y éxito
- **Fondo**: Blanco roto (#FAFAFA) - limpieza y claridad

### Tipografía
- **Títulos**: "Inter" - sans-serif moderna y legible
- **Cuerpo**: "Source Sans Pro" - legibilidad excelente para productos
- **Precios**: "JetBrains Mono" - monoespaciada para números

### Estilo Visual
- Aesthetic industrial moderno con elementos metálicos
- Tarjetas de productos con sombras sutilmente metálicas
- Iconos de herramientas como elementos decorativos
- Fondos con textura sutil de metal cepillado

## Efectos y Animaciones

### Librerías Utilizadas
1. **Anime.js** - animaciones de entrada y transiciones
2. **ECharts.js** - gráficos de estadísticas (si se requieren)
3. **Splide.js** - carrusel de productos destacados
4. **p5.js** - efecto de partículas metálicas en fondo
5. **Matter.js** - simulación de gravedad para productos
6. **PIXI.js** - efectos visuales avanzados
7. **Shader-park** - efectos de metal y reflejos

### Efectos Específicos
- **Hero Section**: Animación de partículas metálicas con p5.js
- **Product Cards**: Hover con elevación y brillo metálico
- **Carrito**: Animación de deslizamiento con contador dinámico
- **Formulario**: Validación en tiempo real con animaciones suaves
- **Fondo**: Patrón sutil de herramientas con shader effects

### Interactividad
- Búsqueda instantánea con filtros por categoría
- Carrito persistente con actualizaciones en tiempo real
- Formulario multi-paso con validación progresiva
- Notificaciones toast para acciones del usuario
- Modal de confirmación para pedidos

### Responsive Design
- Mobile-first approach
- Menú hamburguesa para móviles
- Grid adaptativo para productos
- Touch-friendly para tablets y móviles
